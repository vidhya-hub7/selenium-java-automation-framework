# Selenium Java Automation Framework

## Overview
Automates login, forgot password, and logout flows for a sample web app using Selenium WebDriver (Java) with TestNG assertions.

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven

## Test Coverage
- Invalid login validation (error message)
- Forgot password flow and reset message validation
- Valid login and greeting assertion
- Logout

## How to Run
1. Clone the repo
2. Open in Eclipse/IntelliJ as a Maven project
3. Run the test class: `LoginLogoutAutomationTest`

## Notes
Built as part of hands-on Selenium training and customized with additional assertions and locator strategies.
