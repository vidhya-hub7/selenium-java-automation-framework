# Selenium Java Automation Framework

## Overview
Automated end-to-end testing of critical user flows including login, forgot password, and logout for a web application using Selenium WebDriver with Java and TestNG.

This project demonstrates hands-on experience in functional UI testing, validation handling, and automation of real-world user scenarios.

## Tech Stack
- Java  
- Selenium WebDriver  
- TestNG  
- Maven  

## Test Scenarios Covered
- Invalid login validation with error message verification  
- Forgot password workflow with reset confirmation validation  
- Valid login with successful greeting message assertion  
- Logout functionality verification  

## Key Highlights
- Implemented reusable locator strategies using XPath and CSS selectors  
- Used TestNG assertions for validation of UI elements and messages  
- Automated realistic user flows simulating end-user behavior  
- Structured test cases for better readability and maintainability  

## How to Run
1. Clone the repository  
2. Open the project in Eclipse or IntelliJ as a Maven project  
3. Run the test class: `LoginLogoutAutomationTest`  

## Purpose
This project was built to strengthen practical skills in Selenium automation, test validation, and UI testing workflows aligned with industry practices.
